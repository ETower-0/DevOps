¿Quieres aprender diseño web pero no sabes por dónde empezar?  

¿Cansado de ver tutoriales aburridos y mal explicados?  

Bienvenido al Curso de Diseño Web, curso en el que aprenderás paso por paso y desde lo más básico a diseñar cualquier página web de una forma práctica y sencilla.  

No importa si nunca antes has programado o si no tienes conocimientos de páginas web. Este curso está diseñado para que aprendas desde lo más básico hasta un nivel avanzado en el que podrás diseñar cualquier sitio web de forma profesional.  

Este es más que un curso, una Carrera de Diseño web, porque vas a aprender TODO lo que se necesita para crear cualquier sitio web.  

** Dedica 1 hora al día durante 1 mes con este curso y te aseguro que te convertirás en diseñador web. **  

Iniciamos el curso con una sección en la cual aprenderás a crear tus propios bocetos de sitios web en Adobe XD. Bocetos que más adelante convertiremos en sitios web reales.  

Después veras los fundamentos del diseño web, en donde construirás tu primera página con HTML, y CSS.  

Después aprenderás el arte de convertir tus sitios en increíbles páginas adaptables a dispositivos móviles para que tu página se vea bien desde cualquier dispositivo.  

A continuación aprenderás a programar en Javascript, uno de los lenguajes más importantes. Con el podrás darle lógica a tus páginas para que los usuarios puedan interactuar con ellas.  

Además aprenderás jQuery con el que entre otras muchas cosas aprenderás a agregar efectos y animaciones a tus páginas web.  

Después aprenderás Bootstrap 4, el mejor framework de diseño web con el que podrás hacer sitios web responsive de una forma muy rápida y sencilla.  

Seguiremos con WordPress en donde aprenderás como trabajar con esta maravillosa herramienta como administrador de contenido de tu página web o blog. En este curso aprenderás desde como instalar WordPress hasta como funciona en un sitio real. Incluso al final crearemos un blog real utilizando todo lo aprendido en las secciones anteriores.  

Por ultimo aprenderás como compartir tu código en GitHub mediante el control de versiones Git.  

Y para el final del curso aprenderás como subir tu página web a Internet para que todos puedan acceder a ella.  

Al final del curso serás capas de diseñar cualquier Sitio Web, ya sea el de tu propia empresa, o el de tus clientes. Los beneficios que obtendrás en tu vida con este curso son ilimitados.  

En cada sección del curso hay al menos un proyecto real con la cual vas a poder trabajar con todos los conocimientos adquiridos. Entre los múltiples proyectos que vamos a realizar se encuentran un Portafolio de Trabajos, una página para empresas y/o startups, una página para un Restaurante y un Blog desde el cual podrás ganar ingresos con publicidad.

  • A convertir tú sitio web en un sitio adaptable a dispositivos móviles. (Responsive design).   

  • A programar en JavaScript uno de los lenguajes de programación más importantes.   

  • A manipular tus sitios web para agregarles animaciones y efectos con CSS y jQuery.   

  • A trabajar con Bootstrap para crear sitios web responsive en muy poco tiempo.   

  • A trabajar con WordPress para administrar tus propios sitios web.   

  • A crear un blog de WordPress con un diseño personalizado que haremos.
