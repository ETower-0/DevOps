IP addresses


What is?
The IP address can change often due to changes in the network, or because the device in charge of
assigning IP addresses within the network decides to assign another IP
(for example, with the DHCP protocol ).
This form of IP address assignment is also called a dynamic IP address
(usually abbreviated as dynamic IP ).
